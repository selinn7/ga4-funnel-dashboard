# ga4-funnel-dashboard
# Interactive GA4 e-commerce funnel dashboard (Looker Studio + BigQuery SQL)
# Tech & Tools
Looker Studio · BigQuery · SQL · GA4
# Highlights
- Visual funnel: session → view item → add to cart → checkout → shipping → payment → purchase
- KPIs: Total Sessions, Total Purchases, Conversion Rate
- Interactive filters: date, landing page, source/medium, device, OS, language
- Charts: map, funnel, bar, donut, time series
